Hello Specificity!

Did you see what just happened? The h1 tag was specified with 2 rules in the internal css way and inline style too. What just happened??

Due to the rules of specificity, the rule in the lowermost is applied (most recent). But the color was defined as green in the second one, how come the color became white? This is due to the inline style definition :)

Inline style has highest priority!

Basically, the style values just get overwritten for that property of an element.
